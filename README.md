# Forbes Global Companies Analysis (2011–2023)

## Project Overview
This project analyzes the Forbes Global 2000 companies dataset to uncover
insights into revenue, profits, market value, industry distribution,
and geographic trends. The analysis focuses on understanding global
corporate performance using data-driven techniques.

## Objectives
- Clean and preprocess real-world financial data
- Perform exploratory data analysis (EDA)
- Identify trends, correlations, and outliers
- Generate business-focused insights

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter / Google Colab
- Tableau (for visualization)
